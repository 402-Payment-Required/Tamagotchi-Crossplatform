# Tamagotchi Crossplatform

Cross-platform (iOS/Android/Web) virtual pet app built with Expo + React Native.

## Get started

```bash
pnpm install
pnpm start        # dev server
pnpm run ios / pnpm run android / pnpm run web
```

## Scripts

```bash
pnpm run lint      # eslint + prettier check
pnpm run format    # eslint --fix + prettier --write
pnpm test          # jest
```

See `AGENTS.md` for architecture and conventions.
